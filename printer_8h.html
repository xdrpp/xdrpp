Detailed Description

Support for pretty-printing XDR data types.

Function xdr::xdr_to_string converts an arbitrary XDR data type to a string. In addition, if you say using xdr::operator<<;, you can use the standard operator<< to print XDR types.

You can customize how a particular user-defined type T gets printed by defining a function std::string xdr_printer(const T&) in the same namespace as T.
